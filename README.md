# Uso local

1. Clona este repositorio:

   ```bash
   git clone <URL_DEL_REPOSITORIO>
   ```

2. Instala las dependencias:

   ```bash
   bun i
   ```

3. Inicia el servidor de desarrollo:

   ```bash
   bun dev
   ```

El proyecto estará disponible en [http://localhost:4321](http://localhost:4321).
