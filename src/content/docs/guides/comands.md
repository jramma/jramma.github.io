# Guía Básica de Comandos Unix

Unix es un sistema operativo poderoso, utilizado en todo el mundo en diversas formas. Conocer los comandos básicos de Unix es esencial para cualquier persona que trabaje en tecnología. Esta guía te introducirá a algunos de los comandos más comunes y útiles.

## Comandos Básicos

- **ls**: Lista todos los archivos y directorios en el directorio actual.
- **cd `<directorio>`**: Cambia el directorio actual al especificado.
- **pwd**: Muestra el camino completo del directorio actual.
- **mkdir `<nombre>`**: Crea un nuevo directorio con el nombre especificado.
- **rmdir `<nombre>`**: Elimina un directorio vacío.
- **rm `<archivo>`**: Elimina archivos. Usa `-r` para directorios.
- **cp `<origen>` `<destino>`**: Copia archivos o directorios.
- **mv `<origen>` `<destino>`**: Mueve o renombra archivos o directorios.
- **touch `<archivo>`**: Crea un nuevo archivo vacío o actualiza la marca de tiempo de un archivo existente.
- **cat `<archivo>`**: Muestra el contenido de un archivo.
- **echo `<mensaje>`**: Imprime un mensaje en la terminal.
- **grep `<cadena>` `<archivo>`**: Busca una cadena de texto en un archivo.
- **find `<directorio>` -name `<nombre>`**: Busca archivos y directorios con el nombre especificado.
- **chmod `<permisos>` `<archivo>`**: Cambia los permisos de un archivo o directorio.
- **chown `<usuario>`:`<grupo>` `<archivo>`**: Cambia el propietario y grupo de un archivo o directorio.
